Saving and Loading Sessions

The software allows you to save and load terminal and named pipe window states, so you don't have to reconfigure a session each time you open the application.

Saving Session

  1. Go to File>Save Current Session. A file browser will appear.

  2. Select the location and name of the session file you would like to save to. Then click "Save". Your current session is now saved.

Opening Session

  1. Go to File>Open Previous Session. A file browser will appear.

  2. Select the location and name of the session file you would like to open. Then click "Open". The previous session state will load and automatically initialise, ready to use.
