Considerations

This page outlines a list of a few things to consider when using multiple XBee clients with serial terminals.

Set Coordinator Destination Address on Clients

To acheive optimal performance with clients, it is best to set the destination address of clients to correspond to the Coordinator address. This can be performed in the Digi XCTU software. If this is not set, latency on received packets can occur as routers route around packets. Additionally, other clients can receive the terminal outputs from other clients, which is undesireable.

Baud Rate of Clients

The baud rate of the serial connection on client radio modems can differ, but you should set the baud rate of the coordinator serial connect to be the fastest of all the client speeds for best results.
